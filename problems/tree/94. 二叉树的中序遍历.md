## 题目描述

难度：中等

给定一个二叉树，返回它的中序 遍历。

### 示例1：

```
输入: [1,null,2,3]
   1
    \
     2
    /
   3

输出: [1,3,2]
```

## 解题方法

- 中序遍历的顺序是遍历左子树 - 根节点 - 右子树

```js
var inorderTraversal = function(root) {
    const res = [];
    const helper = (root) => {
        if (!root) {
            return;
        }
        helper(root.left);
        res.push(root.val);
        helper(root.right);
    }
    helper(root);
    return res;
};
```
