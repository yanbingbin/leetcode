## 题目描述

难度：中等

给定一个二叉树，返回它的 后序 遍历。

### 示例1：

```
输入: [1,null,2,3]  
   1
    \
     2
    /
   3 

输出: [3,2,1]
```

## 解题方法

- 后序遍历：按照左子树、右子树、根节点的顺序进行遍历

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var postorderTraversal = function(root) {
    const res = [];
    const helper = (node) => {
        if (node === null) return;
        if (node.left) {
            helper(node.left);
        }
        if (node.right) {
            helper(node.right);
        }
        res.push(node.val);
    }
    helper(root);
    return res;
};
```
